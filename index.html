Aakash is a low cost computing device/tablet for students, the project is initiated and funded by MHRD, Govt. of India. Aakash already runs Android 4.0 with many educational apps made by IIT Bombay. Please refer to androportal for some of the apps.

This is the landing page for the Linux on Aakash documentation. To report any bugs please raise issues.
